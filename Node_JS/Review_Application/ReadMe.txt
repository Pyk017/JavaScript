A Review Application

Requirement:
 - http://localhost:8888/home
 -> Review form
 - http://localhost:8888/review
 -> Display Review

 Implementation:
 - Server = For user to make request.
 - View = Used to display the response of the server.
 - Router = Routers are there to handle different requests depending upon what URL user is going to enter into the browser.
 - Request Handlers = To fulfill the request arrive at the server and that has been routed using the router.
